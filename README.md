# clermontAI

Credit risk and fraud detection ML system for fintech lenders.

## Problem
Current credit scoring relies heavily on bureau data, rejecting ~40% of thin-file borrowers who would have repaid. clermontAI uses behavioral and application-level features with explainable ML (SHAP) to score creditworthiness for borrowers without traditional credit history.

## Status
🚧 In development — Phase 1 (data + EDA)

## Tech Stack
Python, pandas, scikit-learn, XGBoost, SHAP, Streamlit

## Dataset
Home Credit Default Risk (Kaggle) — 307K loan applications, 122 features
